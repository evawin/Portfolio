  • E-commerce Web Development

    This project is a E-Commerce web application developed using HTML5, CSS, JavaScript, Node.js, Express, EJS, and MySQL. The application follows a client–server architecture, where Express handles HTTP GET and POST requests and serves both static and dynamic content. Static assets such as CSS, JavaScript, and images are served from a public directory, while dynamic pages are rendered using EJS templates populated with data retrieved from a MySQL database. User authentication is implemented via a login form that validates predefined credentials before granting access to the store. Product information is stored in a MySQL database and retrieved using parameterised SQL queries to prevent SQL injection. A custom Node.js module is used to encapsulate reusable logic, demonstrating modular application design. Bootstrap 5 is integrated via CDN to ensure a consistent and responsive user interface. Client-side validation ensures required fields are not empty and that valid input is provided, while server-side error handling provides user feedback in cases of authentication failure or database errors.

  • Multi-Router Network Configuration with NAT, DNS and RIP using CISCO Packet Tracer

    Involed the design and configuration of a multi-router network using Cisco Packet Tracer. The network is built using static IP addressing and includes multiple PCs, web servers, and a DNS server connected through six routers. Network Address Translation (NAT) is implemented to map private IP addresses to public addresses, enabling internal users to access external web and DNS services. Routing between all networks is achieved using RIP version 2, allowing dynamic route exchange between routers. A DNS service is configured to provide name resolution for all web servers, and each server hosts a unique web page to verify correct connectivity. The project is successfully validated when all PCs can resolve domain names and access each website using URLs, demonstrating correct implementation of addressing, routing, NAT, DNS, and end-to-end connectivity.

  • Text simplifier application using Java

    This project involved the design and development of a Text Simplifier application and API in Java. The goal of the system is to simplify the language of a given text file by replacing each word with a synonym drawn from the 1,000 most common words in English (Google-1000 list), following the principle of using clear and simple language. The application processes a user-specified text file word by word and dynamically replaces each word with the most semantically similar word from the Google-1000 list. Word similarity is calculated using GloVe word embeddings, which provide numerical vector representations of words based on their semantic meaning.
